# CrisisBoard — Project Management Tool

A full-stack Jira-like project management application built with the MERN stack (MongoDB, Express.js, React, Node.js). Features real-time collaboration, role-based access control, and a drag-and-drop Kanban board.

## Live Demo

Frontend:https://crisisboard-xi.vercel.app
Backend:https://crisisboard-api.onrender.com/health

---

## Features

- JWT authentication with access and refresh tokens
- Role-based access control (Admin, Member, Viewer)
- Workspaces, Projects, Boards, Sprints, and Tasks hierarchy
- Drag and drop Kanban board
- Real-time updates using Socket.io
- Task comments, priorities, labels, and story points
- Responsive dark UI

---

## Tech Stack

Frontend
- React.js (Vite)
- Tailwind CSS
- Zustand (state management)
- React Query (server state)
- Socket.io-client (real-time)
- DnD Kit (drag and drop)
- React Router v6

Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- Socket.io


---
## Deployment

Frontend deployed on Vercel
Backend deployed on Render
Database hosted on MongoDB Atlas
